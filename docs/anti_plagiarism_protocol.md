# Anti-Plagiarism Protocol

1. Commit daily with per-member authorship and clear message prefixes (`M:`, `W1:`, `W2:`, `BIZ:`).
2. Keep experiment outputs timestamped in `reports/` and never backfill metrics manually.
3. Keep all references in `references.md`; quote external ideas with attribution.
4. Keep original diagrams and UI assets in this repo (no stock architecture screenshots).
5. In final video, show a live closed-loop path: `Flag -> local explanation -> user confirmation`.
